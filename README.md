# Usage
As simple as opening the app and pressing a button

**Args:**
  - `-d, --debug`
    - Runs the app with debug mode
  - `-df, --debug-file`
    - Runs the app with debug mode and saves the logs to a temporary file (run `-h` to look at the file path)
  - `-h, --help`
    - Shows the help screen
